## <b>Hitorya-WebServer</b>

### Overview

**This project is a high-performance web server written in C and C++, designed with a focus on maximum efficiency, flexibility, and low-level optimization.** It was built from the ground up without the use of external libraries or frameworks, providing complete control over system resources and behavior.

The server includes custom-designed core components that ensure predictable performance and high reliability under heavy loads.
